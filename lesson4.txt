1. Incorrect way to find users younger than 23
   -- This will scan through all documents
   db.users.find({"age":{$lt:23})

2. Show execution explanation plan of query
db.users.find({"age":{$lt:23}}).explain("executionStats")

3. Create an index on age
   db.users.ensureIndex({"age":1}) -- 1 means true

4. Run explain with index now to see new plan
   db.users.find({"age":{$lt:23}}).explain("executionStats")
   Now it only scanned 2 documents

5. Drop an index
   db.users.dropIndex({"age":1})

6. Display all indexes
   db.users.getIndexes()
   
7. Usually index on fields you search on a lot.
